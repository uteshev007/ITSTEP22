Задачи на урок

Задание 1
Реализуйте класс "Студент".
Необходимо хранить в переменных-членах класса:
 ФИО, дату рождения, контактный телефон, город, страну, название учебного заведения, город и страну (где находится учебное заведение), номер группы.
Реализуйте функции-члены класса для ввода данных, вывода данных, реализуйте аксессоры для доступа к отдельным переменным-членам.


Задание 2
Реализуйте класс "Точка".
Необходимо хранить координаты x, y, z в переменных-членах класса.
Реализуйте функции-члены класса для ввода данных, вывода данных, реализуйте аксессоры для доступа к переменным-членам, реализуйте сохранение в файл и загрузку данных из файла


Задание 3
Реализуйте класс Data с переменными для хранения: день, месяц, год.
Проиллюстрировать работу с этим классом.


Задание 4
Описать класс "ОЗУ" с полями: фирма, частота, объем.
Создать массив экземпляров класса "ОЗУ" и написать программу, позволяющую:
- динамически изменять размер массива;
- выводить список памяти с частотой больше заданной;
- выводить список памяти с объемом больше заданного;
