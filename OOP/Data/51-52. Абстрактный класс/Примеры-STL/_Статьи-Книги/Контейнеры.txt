Контейнеры — это шаблоны классов.
Контейнеры:
1) последовательные контейнеры
	vector - массив с изменением размера
	array - массив с ограниченным размером
	deque - двусторонняя очередь
	list  - двунаправленный список
	forward_list - однонаправленный список
2) ассоциативные контейнеры
	map
	unordered_map 
	set 
	unordered_set
	multimap 
	unordered_multimap
	multiset
	unordered_multiset
3) контейнеры-адаптеры
	queue - FIFO (очередь)
	priority_queue - очередь с приоритетом
	stack - LIFO (последним поступил — первым обслужен)
